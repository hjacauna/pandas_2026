# pandas_2026

Material de estudo e exercícios para o curso **Pandas 2026**, organizado por dias e conjuntos de dados de apoio.

## Estrutura do repositório

- `dia_01/`: exercícios e anotações do primeiro dia.
- `dia_02/`: exercícios e anotações do segundo dia.
- `dia_03/`: exercícios e anotações do terceiro dia.
- `dia_04/`: exercícios e anotações do quarto dia.
- `data/`: arquivos de dados utilizados nos exemplos e atividades.

## Como usar

1. Navegue até a pasta do dia desejado.
2. Abra os notebooks ou scripts para acompanhar os conteúdos.
3. Utilize os arquivos da pasta `data/` conforme necessário.

## Requisitos

- Python 3.10+
- pandas
